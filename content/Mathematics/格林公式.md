# 定理

设平面闭区域 $D$ 由分段光滑的曲线 $L$ 围成，若函数 $P(x,y),Q(x,y)$ 在 $D$ 上具有一阶连续偏导数，则：

$$
\oint_{L}P(x,y)\thinspace \mathrm{d}x+Q(x,y)\thinspace\mathrm{d}y = \iint \limits_{D} \left( \frac{ \partial Q }{ \partial x } -\frac{ \partial P }{ \partial y }  \right) \thinspace \mathrm{d}x\mathrm{d}y
$$

其中 $L$ 是 $D$ 的取正向的边界曲线。
