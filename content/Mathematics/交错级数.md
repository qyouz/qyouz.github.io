# 交错级数审敛法（莱布尼茨定理）

交错级数 $\sum _{n=1}^{\infty}(-1)^{n-1}u_{n}$ 满足条件：

-  $u_{n}>u_{n+1}$
-  $\lim_{ n \to \infty }u_{n}=0$ 

那么交错级数收敛

# 狄利克雷判别法

如果满足以下条件

-  $a_{n}>a_{n+1}$
-  $\lim_{ n \to \infty }a_{n}=0$
- $\left| \sum _{n=1}^{N}b_{n} \right|\leq M$ ，$M$ 是常数。($\sum _{n=1}^{N}b_{n}$ 有界)

则级数 $\sum _{n=1}^{\infty}a_{n}b_{n}$ 收敛

# 阿贝尔判别法

如果满足以下条件

-  $a_{n}>a_{n+1}$
-  $\lim_{ n \to \infty }a_{n}=M$ ，$M$ 是常数
- $\sum _{n=1}^{\infty}b_{n}$ 收敛 

则级数 $\sum _{n=1}^{\infty}a_{n}b_{n}$ 收敛
