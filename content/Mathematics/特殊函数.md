# 矩形函数(Rectangular function)

$$
\text{rect}(t)=\Pi(t)=
\begin{cases}
0 &\text{ if }|t|> \frac{1}{2} \\
\frac{1}{2} &\text{ if }|t|= \frac{1}{2} \\
1 &\text{ if }|t| < \frac{1}{2}
\end{cases} \text{ or }\begin{cases}
1 & \text{ if }|t| < \frac{1}{2} \\
0 & \text{ if }|t| > \frac{1}{2}  
\end{cases}
$$

# 三角形函数(Triangular function)

$$
\text{tri}(t) = \Lambda(t) = \begin{cases}
1-|t|& |t| < 1 \\
0  & \text{otherwise}
\end{cases}
$$

或者是两个相同的单位矩形函数的卷积：

$$
\text{tri}(t) = \text{rect}(t) * \text{retc}(t)
$$

# 狄拉克 $\delta$ 函数(Dirac delta function)

$$
\delta(t) = \begin{cases}
+\infty , & x =0 \\
0 , & x \ne 0
\end{cases}  \text{ , } \int_{-\infty}^{+\infty} \delta(t) \thinspace \mathrm{d}x =1
$$

# Sigmoid 函数

$$
\sigma(x) = \frac{1}{1+e^{-x}} = \frac{e^x}{1+e^x} = 1- \sigma(-x)
$$

# 符号函数 (Sign function)

$$
\text{sgn}(x) = \begin{cases} 1 & \text{if } x > 0 \\ 0 & \text{if } x = 0 \\ -1 & \text{if } x < 0 \end{cases}
$$
