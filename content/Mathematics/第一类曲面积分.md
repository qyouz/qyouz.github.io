# 对面积的曲面积分

> [!NOTE] 
 > 计算三维空间中有密度的曲面的质量
 >

曲面 $\Sigma$ :

$$
z =z(x,y)
$$

$f(x,y,z)$ 在曲面 $\Sigma$ 上的积分：

$$
\iint \limits_{ \Sigma} f(x,y,z)\thinspace\mathrm{d}S = \iint \limits_{ D} f[x,y,z(x,y)]\sqrt{ 1+ \left( \frac{ \partial z }{ \partial x }  \right)^{2}+\left( \frac{ \partial z }{ \partial y }  \right) } \thinspace \mathrm{d}x\mathrm{d}y
$$

**要点** ：把对面积的曲面积分化为**二重积分**

把三维曲面投影到二维平面，微元的转换：

[雅可比行列式](雅可比行列式.md)

$$
\mathrm{d}S = \sqrt{ 1+ \left( \frac{ \partial z }{ \partial x }  \right)^{2}+\left( \frac{ \partial z }{ \partial y }  \right) } \thinspace \mathrm{d}x\mathrm{d}y
$$
