# 定理

设空间闭区域 $\Omega$ 是由分片光滑的闭曲面 $\Sigma$ 所围成，若函数 $P(x,y,z),Q(x,y,z),R(x,y,z)$ 在 $\Omega$ 上具有一阶连续偏导数，则有：

$$
\iiint \limits_{ \Omega} \left( \frac{ \partial P }{ \partial x } +\frac{ \partial Q }{ \partial y } + \frac{ \partial R }{ \partial z }  \right) \mathrm{d}v = \iint \limits_{ \Sigma} P\thinspace\mathrm{d}y\mathrm{dz}+Q\thinspace\mathrm{d}z\mathrm{d}x+R\thinspace\mathrm{d}x\mathrm{d}y
$$

这里 $\Sigma$ 是 $\Omega$ 的整个边界曲面的外侧。
