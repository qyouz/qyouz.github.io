# 指数函数

$$
e^z = e^{x+iy} = e^x(\cos y + i \sin y)
$$

- **周期性**: $e^{z+2k\pi i} = e^z, \quad k \in \mathbb{Z}$
- **导数**: $\frac{d}{dz} e^z = e^z$
- **模与幅角**: $|e^z| = e^x, \quad \text{Arg}(e^z) = y + 2k\pi$

# 对数函数

$$
\text{Ln } z = \ln|z| + i \arg z = \ln|z| + i(\text{Arg } z + 2k\pi), \quad (z \neq 0)
$$

- **主值**: $\ln z = \ln|z| + i\text{Arg } z, \quad (-\pi < \text{Arg } z \le \pi)$
- **性质**: $\text{Ln}(z_1 z_2) = \text{Ln } z_1 + \text{Ln } z_2$

# 幂函数

$$
w = z^\alpha = e^{\alpha \text{Ln } z}
$$

- **多值性**:
    1. $\alpha = n \in \mathbb{Z}$: 单值
    2. $\alpha = \frac{m}{n}$ (最简分数): $n$ 个值
    3. $\alpha$ 为无理数或复数: 无穷多值

# 三角函数

$$
\sin z = \frac{e^{iz} - e^{-iz}}{2i}, \quad \cos z = \frac{e^{iz} + e^{-iz}}{2}
$$

$$
\tan z = \frac{\sin z}{\cos z}, \quad \cot z = \frac{\cos z}{\sin z}
$$

- **常用公式**:
    - $\sin^2 z + \cos^2 z = 1$
    - $\sin(z_1 \pm z_2) = \sin z_1 \cos z_2 \pm \cos z_1 \sin z_2$
    - $|\sin z|^2 = \sin^2 x + \sinh^2 y$
    - $|\cos z|^2 = \cos^2 x + \sinh^2 y$

# 反三角函数

$$
\text{Arcsin } z = -i \text{Ln}(iz + \sqrt{1-z^2})
$$

$$
\text{Arccos } z = -i \text{Ln}(z + \sqrt{z^2-1})
$$

$$
\text{Arctan } z = \frac{i}{2} \text{Ln} \frac{i+z}{i-z}
$$

# 双曲函数与反双曲函数

$$
\sinh z = \frac{e^z - e^{-z}}{2}, \quad \cosh z = \frac{e^z + e^{-z}}{2}
$$

- **复数域联系**:
    - $\sin(iz) = i \sinh z, \quad \cos(iz) = \cosh z$
    - $\sinh(iz) = i \sin z, \quad \cosh(iz) = \cos z$
- **反双曲函数**:
    - $\text{Arsh } z = \text{Ln}(z + \sqrt{z^2+1})$
    - $\text{Arch } z = \text{Ln}(z + \sqrt{z^2-1})$
